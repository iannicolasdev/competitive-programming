# Matriz

Um vetor pode ser de inteiros, de números decimais `(float)`, mas também podemos criar um vetor de vetores, que também pode ser chamado de matriz.

Para definir uma matriz devemos usar o tipo de dados que essa matriz vai receber juntamente com o nome da variável atribuida a ela e a quantidade de linhas e colunas.

```cpp
int matriz[N][M];
```
